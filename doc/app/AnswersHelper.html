    my_vote(answer) click to toggle source

    Given an answer, returns the vote on it that belongs to the current user, or nil if there isn't one. Simply saves having to clutter up the view getting hold of it every time (which is sort of the point of all helpers, I guess).

    # File app/helpers/answers_helper.rb, line 6
def my_vote(answer)
  if user_signed_in?
    return answer.votes.where(:user => current_user).first
  end
  return nil
end
